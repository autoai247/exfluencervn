# 관리자 계정 정보

## 접속 방법

### 관리자 로그인 페이지
```
http://localhost:3001/auth/admin/login
```

### 데모 관리자 계정 (테스트용)
```
이메일: admin@exfluencervn.com
비밀번호: admin123
```

## 관리자 기능

### 1. 대시보드 개요
- 전체 사용자 통계 (인플루언서, 광고주)
- 캠페인 현황 (전체, 활성, 대기)
- 거래액 통계
- 알림 및 대기 항목

### 2. 사용자 관리
- 전체 사용자 목록 조회
- 사용자 검색 및 필터링
- 계정 상태 관리 (활성, 정지, 차단)
- 사용자 상세 정보 확인
- 계정 정지/해제

### 3. 캠페인 승인
- 신규 캠페인 검토
- 캠페인 승인/거부
- 진행 중인 캠페인 모니터링
- 캠페인 상세 정보 확인

### 4. 출금 관리
- 출금 요청 목록
- 출금 승인/거부
- 은행 정보 확인
- 송금 처리 관리

## 관리자 권한

- ✅ 모든 사용자 정보 조회
- ✅ 계정 정지/차단
- ✅ 캠페인 승인/거부
- ✅ 출금 요청 처리
- ✅ 시스템 통계 조회
- ✅ 신고 사용자 관리

## 보안 사항

⚠️ **중요**: 관리자 로그인 URL은 공개하지 마세요!
- `/auth/admin/login`은 숨겨진 URL입니다
- 일반 사용자는 이 URL을 모릅니다
- 모든 관리자 활동은 로그로 기록됩니다

## 실제 배포 시 주의사항

1. **비밀번호 변경**: 반드시 강력한 비밀번호로 변경
2. **2단계 인증**: 가능하면 2FA 추가
3. **IP 제한**: 특정 IP에서만 접속 허용
4. **활동 로그**: 모든 관리자 활동 기록
5. **권한 분리**: 필요시 슈퍼관리자/일반관리자 구분

## 페이지 구조

```
/auth/admin/login          → 관리자 로그인
/main/admin                → 관리자 대시보드
  ├─ 개요 탭              → 전체 통계 및 알림
  ├─ 사용자 관리 탭       → 사용자 목록 및 관리
  ├─ 캠페인 승인 탭       → 캠페인 검토 및 승인
  └─ 출금 관리 탭        → 출금 요청 처리
```
